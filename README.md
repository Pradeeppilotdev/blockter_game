## ✨ Project Snapshot
Blockter is an electrifying, fast-paced block manipulation game inspired by classic arcade action and modern audiovisual design. Navigate procedurally generated arenas, chain combos, and outmaneuver adaptive obstacles as the environment morphs to the rhythm.

- Genre: Arcade / Puzzle / Action
- Experience: Single-player (local)
- Mood: Neon, cyberpunk, synthwave
- Target platforms: Desktop / Web (engine-agnostic assets and docs)

---

## 🚀 Features (v1.0)
- Dynamic, rhythm-synced levels that evolve with player input
- Modular block types: Reflective, Phantom, Magnet, Time-Slip
- Procedural stage generator with escalating difficulty curves
- Combo & multiplier systems for high-score chasing
- Visual effects: bloom, scanlines, chromatic aberration (configurable)
- Accessibility modes: color-blind palettes and adaptive speed

---

## 🕹️ How to Play
- Objective: Clear or manipulate blocks to survive each wave and score max points.
- Basic controls (default):
  - Move / Aim: Arrow keys / WASD / Gamepad left stick
  - Action / Interact: Space / Enter / Gamepad A
  - Slow-time (if equipped): Shift / Left Trigger
  - Toggle HUD / Visuals: H / Y

Tip: Chain different block reactions (e.g., Magnet → Reflective → Time-Slip) to build massive multipliers.
